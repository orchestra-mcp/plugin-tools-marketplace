# Orchestra Plugin: tools-marketplace

A tools plugin for the [Orchestra MCP](https://github.com/orchestra-mcp/framework) framework.

## Install

```bash
go install github.com/orchestra-mcp/plugin-tools-marketplace/cmd@latest
```

## Usage

Add to your `plugins.yaml`:

```yaml
- id: tools.tools-marketplace
  binary: ./bin/tools-marketplace
  enabled: true
```
